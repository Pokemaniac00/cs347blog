My first week and a half of Web Development has given me the opportunity to practice and understand HTML on a higher level than when I performed my own personal research to learn HTML. I have enjoyed the in-class labs so far, and I wanted to share some of my more recent personal research on HTML in this week's blog entry. My HTML files have seen much more structure like in the code snippet below (adding a code snippet was also something I researched for this week). 

            
                <!DOCTYPE html>
                <html>
                    <head>
                        <title>Document</title>
                    </head>
                    <body>
                        <header>
                            <h1>Welcome to the website!</h1>
                        </header>
                        <main>
                            <p>psst! Check the browser to see a pacman pizza!</p>
                            <p>Did you know Pacman's original design was inspired from a pizza with a slice taken away?</p>
                        </main>
                        <footer>
                            Contact us!
                        </footer>
                    </body>
                </html>

The code snippet above was able to be added using encodings for the characters '<' and '>'. These encodings begin with a '&' and are followed up with 'lt;' or 'gt;' depending on which side of bracket you need. This can be useful to teach others about HTML from within an HTML page! There are many online converters as well to make the HTML tags readable on pages.
