Instead, I decided to go for a simple JS program that counts the number of vowels in an input text box, using the javascript code below. I was inspired on making a simple JS program since we were recently introduced to JS. A second inspiration was in this youtube video. To accomplish this task in a minimum amount of lines, I researched how to use regular expressions in javascript. I used a regular expression /[aeiou]/gi to create an array of case insensitive vowels in a string. I then return the length and alert the page with the number of vowels. 

                
function getInput() {
    let text = document.getElementById("vowels").value;
    const count = text.match(/[aeiou]/gi).length;
    alert(count);
}
