This week, I decided to make a to-do list using javascript. This project was inspired by the github repository here. I wanted more opportunities to see some of what Javascript has to offer as we venture forward into node.js. One day, I would like to adapt code like this to create a blog where users can make their own posts and register for accounts. I believe that creating a website like that would be a great long-term goal. In the meantime, click this text here to make a funny sound!

You can add to the list by typing in the input box, then pressing the '+' button. When you press the button, the item is added to the list. Click the item in the list once to cross it off, and click it twice to remove it from the list. 

                
let add = document.getElementById('add');
let list = document.getElementById('list');
let input = document.getElementById('input');
let text = document.getElementById('thatsPrettyGood');

add.addEventListener('click', function() {
    var paragraph = document.createElement('p');
    paragraph.classList.add('paragraph');
    paragraph.innerText = input.value;
    list.appendChild(paragraph);
    input.value = '';
    paragraph.addEventListener('click', function() {
        paragraph.style.textDecoration = 'line-through';
    });
    paragraph.addEventListener('dblclick', function() {
        list.removeChild(paragraph);
    });
});

text.addEventListener('click', function() {
    var audio = new Audio("hey-thats-pretty-good.mp3");
    audio.play();
});
